# ![](LOGO/logo.png "logo") 19 Antique 中古世紀古董網站 

此專案為「靜態切版」，部分功能有使用 javascript 及 jQuery 實作 (如:每日簽到抽獎等)

## 網站連結
https://hangineer.github.io/Antique-Website/

## 使用技術
HTML + CSS + Bootstrap + Javascript + jQuery

<br>

## 網站架構

1. 首頁
   - 每日簽到
   - 輪播圖
   - 商品展示
   - 註腳
   - 回到頂部按鈕
2. 會員中心
   - 登入
3. 商品詳細頁面
   - 不同商品款式的呈現
   - 數量選擇
4. 關於我們
